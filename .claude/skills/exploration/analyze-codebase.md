# Skill: Analyze Codebase Structure

## Purpose
Produce a comprehensive map of a code area including files, dependencies, data flows, and patterns.

## Steps

1. **Broad Search** — Keywords, file listings, module docs
2. **Dependency Tracing** — Imports, shared types, circular deps
3. **Pattern Discovery** — Error handling, naming, abstractions, config access
4. **Data Flow Mapping** — Input → transformation → output → storage

## Output
- File inventory with purposes
- Dependency graph
- Key patterns and conventions
- Recommendations for next agent
